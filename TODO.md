## 1. 산업 통계/리포트 수집 (P-4, U-1)

> 인터뷰 질문 설계, 시장규모 산정, 문제 정의의 기초 데이터

- [x] 국내 제조업체 수 — 537,580개 (2023 CSV), 504,728개 (2024 잠정)
- [x] 제조업 종사자규모별 분포 — 10~299명: 77,389개 (14.39%)
- [x] 제조업 중분류 업종별 전체 — C25~C31 도면 핵심 7개 업종: 237,127개 (44.1%)
- [x] 업종별 CAD/도면 사용 분류 — A(강타겟)/B(혼합)/C(후순위) 구분 완료
- [x] 중소 제조업 PLM 도입률 — 스마트공장 지원DB 기준 4.4%
- [x] 클라우드/SaaS 수용성 — 제조업(10인+) 클라우드 이용률 73.9%
- [x] 글로벌 PLM 시장 규모 — $27.9B~$46.3B (2025), CAGR 7.5~9.1%
- [x] 국내 PLM/PDM 시장 규모 — 약 4,000~5,000억 원 (업계 추정)
- 정리 위치: [[40_타겟/시장규모-산정]]

### 사전 준비

- [x] 인터뷰 질문지 설계 → [[40_타겟/고객-인터뷰-질문지]]
- [ ] 인터뷰 대상 섭외
