# 영업 인터뷰 1차 (2026-02-25 ~ 02-27)

화성 소공인 업체 대상 콜드콜 기반 인터뷰.
목적: PLM/BOM 관리 현황 파악 및 시제품 사용 의향 확인.

## 1. 요약

| 항목 | 수치 |
|------|------|
| 총 콜 수 | 36건+ |
| 인터뷰 성공 | 6건 |
| 실패 (바로 끊음/부재중/CS직원) | 30건+ |
| 시제품 사용 의향 | 1건 (#25 두원이엔지) |

### 인터뷰 대상 프로파일

| 항목 | 값 |
|------|---|
| 지역 | 경기도 화성시, 김포시, 시흥시 |
| 업종 | 금형, 기계제작, 레이저가공, CNC, 반도체장비 등 |
| 규모 | 대부분 영세기업~소기업 (1~9명 추정) |
| 인터뷰 방식 | 콜드콜 (전화) |

| 항목 | 값 |
|------|---|
| 업체명 | 동광기계 |
| 도메인 | 기계제작 (압출기/분쇄기/파쇄기) |
| 위치 | 경기도 화성시 봉담읍 |
| 날짜 | 2026-02-25 |

**Q&A:**
1. PLM / 도면관리 + BOM 관리 소프트웨어를 사용하시나?
   - 영세기업이라 사용하지 않음
2. 그렇다면 자재 관리는 어떻게 하시나?
   - 기업별로 폴더를 만들어서 관리함
   - 영세기업이라 관리할 capa가 나오지도 않음
3. 관리하기 용이한 솔루션을 만들고 있는데 무료 시제품을 사용하실 생각이 있으신가?
   - 곧 폐업한다고 함

**태그:** `PLM 미사용` `폴더관리` `관리 여력 없음` `폐업 예정`

### 4.3 타겟 미스매치 시사점

이번 인터뷰 대상은 대부분 **1~9명 영세기업/소공인**이었음.
- "영세기업이라 관리할 capa가 나오지도 않음" (#11)
- "규모가 작아 사용하지 않는다" (#23)
- "큰 데다 문의해봐라" (#22)
- "남의 제품을 가공하니 자재관리에 신경을 쓸 여력이 없다" (#25)

시장규모-산정.md에서 정의한 타겟(10~299명)과 실제 인터뷰 대상(1~9명)이 불일치.
**다음 인터뷰는 10~49명 규모 기업을 타겟해야 함.**
