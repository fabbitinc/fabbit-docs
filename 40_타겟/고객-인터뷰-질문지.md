# 고객 인터뷰 질문지

예비창업패키지 사업계획서 Problem/Solution 근거 수집용.
인터뷰 소요시간: 약 40~50분.

## 인터뷰 전 체크

- [ ] 녹취 동의 확인
- [ ] 응답자 기본 정보 기록 (아래 0번)
- [ ] 이 질문지 출력 또는 태블릿 준비
- [ ] 대상이 우선 업종(C25/C29/C28)인지 확인
- [ ] 5~9명 구간 기업 최소 1~2곳 포함 (SAM 검증용)

## 0. 기본 정보 (2분)

> 인터뷰 대상 우선 업종: C25(금속가공), C29(기계장비), C28(전기장비)

| 항목 | 기록 |
|------|------|
| 회사명 | |
| 업종 (ex. 금속가공, 기계부품, 전기장비) | |
| 종사자 수 | ☐ 1~4명 ☐ 5~9명 ☐ 10~49명 ☐ 50~299명 ☐ 300명+ |
| 응답자 역할 | |
| 월간 도면 취급 건수 (대략) | |
| 현재 유료 클라우드 서비스 사용 여부 | (ex. ERP, 그룹웨어, 회계 등 아무거나) |

## 1. 현재 업무 프로세스 (10분)

도면과 BOM을 중심으로 일상 업무 흐름을 파악한다.

### 공통

> "도면을 받거나 만들고 나서, 그 다음에 어떤 일이 일어나는지 순서대로 설명해주세요."

- 도면은 어떤 형식으로 관리하나요? (CAD 파일, PDF, 종이 등)
- 도면을 어디에 저장하나요? (로컬PC, 공유폴더, NAS, 클라우드, 종이)
- BOM은 어떻게 작성하나요? (엑셀 수동입력, ERP 연동, CAD 추출 등)
- BOM 작성은 누가 하나요? (설계자 본인, 생산팀, 별도 담당자)

### 수치 확인

| 측정 항목 | 응답 |
|----------|------|
| 도면 1건당 BOM 작성 소요시간 | __분 |
| 월간 신규/변경 도면 건수 | __건 |
| BOM 작성에 투입되는 월간 총 시간 | __시간 |

## 2. 문제와 고충 (15분)

Pain Point를 깊이 파고든다. **응답 원문을 그대로 기록할 것.**

### 도면 관리

- 필요한 도면을 찾을 때 보통 얼마나 걸리나요?
- 도면 버전이 헷갈려서 문제가 된 적 있나요? 구체적으로 어떤 일이 있었나요?
- 설계 변경이 생기면 관련 부서에 어떻게 전달하나요? 얼마나 걸리나요?

| 측정 항목 | 응답 |
|----------|------|
| 도면 검색 소요시간 (건당) | __분 |
| 최근 6개월 내 버전 혼선 발생 횟수 | __회 |
| 설계변경 통보 → 반영 완료 리드타임 | __일 |

### BOM 오류

- BOM에서 실수가 발견된 적 있나요? 어떤 종류의 실수였나요?
- 그 오류 때문에 실제로 어떤 일이 발생했나요? (재발주, 생산 중단, 납기 지연 등)
- 오류를 발견하고 수정하는 데 보통 얼마나 걸리나요?

| 측정 항목 | 응답 |
|----------|------|
| 월간 BOM 오류/재작업 발생 건수 | __건 |
| 오류 1건당 수정 소요시간 | __시간 |
| 오류로 인한 최대 금전 손실 (추정) | __만원 |

### 핵심 질문

> "도면이나 BOM 관리에서 **가장 짜증나는 순간**이 언제인가요?"

> "지금 방식에서 **절대 바뀌지 않을 것 같은 문제**가 있다면 뭔가요?"

## 3. 기존 대안 경험 (5분)

### PLM/PDM 사용 경험

- PLM이나 PDM이라는 솔루션을 들어본 적 있나요?
- 사용해본 적 있다면 → 어떤 제품이었나요? 왜 안 쓰게 됐나요?
- 사용해본 적 없다면 → 도입을 검토한 적 있나요? 안 한 이유가 뭔가요?

## 4. 솔루션 반응 (10분)

> 여기서 Fabbit 데모 또는 컨셉 화면을 보여준다.
> "도면을 업로드하면 AI가 표제란과 부품표를 인식해서 BOM을 자동 생성하는 서비스입니다."

- (보여준 후) 첫 느낌이 어떤가요?
- 이 기능이 있었으면 지금 업무에서 뭐가 달라질 것 같나요?
- 부족하거나 추가로 필요한 기능이 있나요?
- 이걸 실제로 쓸 것 같나요? 쓴다면 / 안 쓴다면 이유는?

### 반응 수치

| 항목 | 응답 |
|------|------|
| 도입 의향 (1~5점) | |
| 추천 의향 (1~5점) | |
| 예상 시간 절감률 (%) | |

## 5. 가격/도입 의향 (5분)

- 이런 서비스에 월 얼마까지 낼 수 있나요?
- 구매 의사결정은 누가 하나요? (본인, 팀장, 대표 등)
- 도입을 결정할 때 가장 중요한 기준은 뭔가요? (가격, 기능, 보안, 기존 시스템 연동 등)
- 무료 파일럿을 제안하면 참여하시겠나요?

| 항목 | 응답 |
|------|------|
| 월 지불 의향 가격 | __만원 |
| 의사결정자 | |
| 도입 최우선 기준 | |
| 파일럿 참여 의향 | Y / N |

## 6. 마무리 (2분)

- 혹시 제가 물어보지 않은 것 중에, 도면이나 BOM 관리에서 하고 싶은 말이 있나요?
- 주변에 비슷한 고민을 하는 분을 소개해주실 수 있나요?
