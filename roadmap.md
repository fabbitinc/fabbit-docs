# Fabbit 개발 로드맵

> "도면(Drawing)과 자재내역(BOM)이 따로 놀아 발생하는 버전 관리 실패 및 제조 비용 손실"을 해결하는 차세대 AI 기반 초경량 PLM

# 📅 Phase 0: 준비 및 검증 (현재 ~ 2026년 3월)

**목표:** 예창패 합격 및 시장 수요 데이터 확보

- **디지털 수요 검증:** 랜딩 페이지를 구축하여 '베타 테스터' 신청을 받고, 그 숫자를 사업계획서에 '시장 수요 증거'로 제시합니다.
- **문제 실체화:** 온라인 커뮤니티(캐드/설계 카페 등) 설문을 통해 "도면-BOM 불일치로 인한 오발주 비용" 등 구체적인 고통 지수를 수집합니다.
- **사업계획서 고도화:** 사용자님의 백엔드/DevOps 전문성과 현재 완성된 MVP UI를 결합하여 '기술력'과 '실행력'을 동시에 증명합니다.

# 🚀 Phase 1: PDM 코어 구축 (2026년 상반기 ~ 하반기)

**목표:** "엑셀 노가다"를 없애는 독보적인 사용성 확보 (MVP)

- **AI 파싱 고도화:** 다양한 양식의 도면 표제란(Title Block)과 엑셀 리스트를 90% 이상의 정확도로 인식하는 모델 최적화.
- **초고속 웹 뷰어:** 설치 없이 웹에서 대용량 도면을 즉시 열람하고 확대/축소 가능한 뷰어 제공.
- **외부 공유 링크:** 로그인 없이 협력사나 고객사에 도면을 안전하게 공유할 수 있는 뷰어 전용 URL 생성 (Viral Loop 유도).
- **버전 관리 자동화:** 같은 품번 업로드 시 리비전 자동 증가 및 구형 도면 "폐기됨" 워터마크 자동 적용.
- **Zero-Interference 매칭:** 고객사의 기존 엑셀 컬럼명을 그대로 유지하면서 jsonb 필드에 매핑하는 유연한 데이터 구조 완성.
- **출시 전략:** Starter(월 24.9만 원) 요금제 런칭, 초기 3~5개 파트너사 확보를 통한 AI 학습 데이터 수집.

# 📈 Phase 2: PLM 워크플로우 확장 (2027년)

**목표:** 데이터의 "신뢰성"과 "협업" 가치 증명 (Commercialization)

- **시각적 협업:** 도면 내 특정 좌표(X, Y)에 댓글 달기, @멘션 기능으로 직관적인 커뮤니케이션 지원.
- **리비전 관리:** 도면 버전별 차이점(Diff)을 시각적으로 비교하고 변경 히스토리를 추적하는 기능.
- **모바일 뷰어:** 현장에서 모바일 기기로 도면을 즉시 확인하고 검토할 수 있는 전용 뷰어 모드.
- **승인 시스템:** '검토-승인-확정'으로 이어지는 모바일 알림톡 연동 간편 결재 프로세스.
- **API 개방:** ERP, MES 등 사내 레거시 시스템과 데이터를 원활하게 연동할 수 있는 표준 API 및 웹훅(Webhook) 지원.
- **수익 확대:** 리비전과 워크플로우 기능이 포함된 Professional 티어 런칭.

# 🌐 Phase 3: 통합 제조 데이터 허브 (2028년)

**목표:** ERP/MES 연동을 넘어선 플랫폼화 (Expansion)

- **ERP/MES 연동 패키지:** 주요 국산/외산 ERP와 데이터를 직접 연동하는 표준 인터페이스 제공.
- **지능형 예측 AI:** 누적된 BOM 데이터를 분석하여 원가 변동을 예측하거나 대체 부품을 추천.
- **경량 MES 확장:** 도면에 QR 코드를 부여하여 현장 생산 현황(공정 진척도)을 실시간으로 추적.
- **공급망 포털:** 협력사와 도면 및 발주 데이터를 안전하게 공유하는 Supplier Portal 기능.
- **Elite 티어:** 월 120만 원 이상의 기업용 커스텀 솔루션 시장 진입.

### 가격 정책 프레임워크

- **Starter (월 24.9만 원):** 소규모 팀을 위한 핵심 PDM 기능 (Phase 1)
- **Professional (별도 협의):** 워크플로우 및 심화 협업 기능 (Phase 2)
- **Elite (월 120만 원~):** ERP/MES 연동 및 커스텀 기술 지원 (Phase 3)

# 🐰 Fabbit 성공을 위한 핵심 지표 (KPI)

- **Time-to-Value:** 사용자가 첫 도면을 던지고 eBOM을 생성하기까지 걸리는 시간 (목표: 1분 이내).
- **Data Integrity:** AI가 추출한 데이터와 실제 도면의 일치율 (목표: 99% 이상, 검토 단계 포함).
- **Active Users:** 주간 단위로 새로운 프로젝트를 생성하거나 도면을 승인하는 관리자 수.
