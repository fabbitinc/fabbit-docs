## 1. 문제인식

> 평가: "창업과제 배경 및 필요성, 목표시장(고객) 현황 분석"

### 해결하려는 문제

**중소 제조업의 제조 데이터가 엑셀·폴더·담당자 머릿속에 갇혀 있어, 검색·변경추적·의사결정이 불가능하다.**

국내 제조업 537,580개 사업체 중 10~299명 규모의 중소기업은 77,389개(14.4%)이며 (통계청 전국사업체조사, 2023), 이 중 **95.6%가 PLM(제품 수명주기 관리) 없이** 도면과 BOM(부품표)을 관리하고 있다 (KDI 스마트공장 보고서, 2014~2017 지원기업 PLM 도입 비중 4.4%). 도면은 폴더에, BOM은 엑셀에, 공급사 정보는 담당자 머릿속에 흩어져 있다.

| 현장 문제           | 정량 근거                                                                                         |
| ------------------- | ------------------------------------------------------------------------------------------------- |
| 도면/BOM 수동 입력  | 엔지니어 1인당 연간 52~260시간 엑셀 BOM 작업에 소요 (OpenBOM, "The Real Cost of Managing BOMs in Spreadsheets") |
| 도면 검색·버전 혼선 | 데이터 전문가가 주당 업무시간의 **30%를 데이터 검색·준비**에 소비, 20%를 중복 작업에 소비 (IDC 연구, Siemens 인용). 지식노동자 기준 25~30% (AIIM; McKinsey). 제조 엔지니어도 도면·부품 정보 검색에 동일 패턴 |
| BOM 오류 → 재발주   | BOM 오류가 전체 설계변경(ECO)의 최대 30% 차지, 건당 행정비 $1,500~3,000 (Aberdeen Group, CADTALK·OpenBOM 재인용) |
| 변경 영향 파악 불가 | 부품 하나를 바꿔도 어떤 제품·공급사·공정에 영향이 가는지 추적할 수 없음                           |

정부 스마트공장 실태조사 (중기부, 2024.10~2025.01)에서 ERP(76.3%), MES(14.4%)는 도입률을 집계하지만, **PLM은 조사 항목 자체에 포함되지 않는다.** 제조 데이터 관리가 시스템이 아닌 사람에게 의존하고 있으며, 이 데이터가 구조화되지 않는 한 AI를 활용한 의사결정 지원도 불가능하다.

### 문제의 시급성

- **제조 현장 고령화**: 숙련공 퇴직 시 머릿속 지식(어떤 부품이 어디에 쓰이는지, 어떤 공급사가 신뢰할 수 있는지)이 함께 사라진다. 데이터가 시스템에 없으면 복구 불가.
- **공급망 불안정**: 미중 갈등(관세·수출규제), 소재 가격 변동으로 부품 대체·공급사 전환이 빈번해졌으나, 변경 영향을 추적할 수 없어 대응이 늦다.
- **AI 도입 전제조건 미충족**: OECD 기준 기업 AI 사용률이 급증(2023년 8.7% → 2025년 20.2%, OECD 통계)하지만, 국내 제조업 AI 도입률은 3.9% (한국생산성본부 산업디지털전환실태조사, 2024)에 불과하다. 데이터가 구조화되지 않으면 AI는 쓸 수 없다.

### 착안 배경

PLM 솔루션 기업에서 BOM 관리 모듈을 직접 개발하며, 중소 제조업 현장의 문제를 목격했다. 고객은 "빠른 도입과 쉬운 사용"을 원했지만, 고객사별 커스텀 개발이 반복되면서 제품은 복잡해지고 도입 장벽은 높아졌다. 근본 원인은 **데이터가 구조화되지 않은 상태에서 관리 기능만 얹으려 한 것**이었다.

AI 도면 분석 스타트업(비링커)이 Seed+Pre-A 누적 투자 10억 이상을 유치하고 글로벌 팁스에 선정(3년 최대 6억)되며 **AI로 제조 데이터를 추출하는 시장의 존재를 입증**했다. 그러나 비링커는 도면에서 데이터를 추출(→Excel)하는 데 그치며, 추출된 데이터를 연결하여 활용하는 단계로는 나아가지 않았다. **"AI로 데이터를 쌓고, 쌓인 데이터에서 AI가 스스로 위험을 감지하고 행동하는" 플랫폼 영역에 진입 기회**가 열려 있다.

### 목표시장 분석

**고객**: 종사자 10~49명, 매출 10억 이상, 금속가공(C25)/기계장비(C29)/전기장비(C28) 중소 제조업

> 평가: "창업과제 현황, 과제 실현 및 구체화 방안"

**MVP 개발 완료 (시제품 단계)** — 핵심 기능이 구현되어 동작하는 상태이며, 파일럿 테스트 및 고도화를 준비 중이다.

| 항목        | 현황                                                                        |
| ----------- | --------------------------------------------------------------------------- |
| AI/PLM 코어 | 도면·BOM AI 추출, 구조화 BOM 관리, 변경관리, 온톨로지 기반 데이터 구조 구현 |
| 프론트엔드  | 대시보드, 부품/BOM, 이슈/변경요청, 조직관리 등 구현                         |
| 인프라      | 1인 풀스택 구축 (서버, 웹, DWG 변환, 스토리지, 그래프 DB)                   |

{이미지 첨부 - 제품 스크린샷 2~3장: (1) 도면 AI 인식 결과 화면, (2) BOM 트리 관리 화면, (3) 대시보드 또는 이슈/변경요청 화면}

### 핵심 기술 및 경쟁 우위

#### 1단계 — AI 자동 추출로 데이터화 (진입 장벽 해소)

중소 제조업이 시스템을 도입하지 못하는 가장 큰 이유는 **기존 데이터를 옮기는 비용**이다. Fabbit은 AI가 이 과정을 대신한다.

- **BOM 자동 분석**: 도면(DWG/PDF) 업로드 시 LLM 기반 AI가 부품표를 자동 인식하여 품명/재질/수량/규격을 추출, 시스템 내 구조화 BOM으로 자동 등록 (현재 구현 완료)
- **도면 자동 인식 (고도화 예정)**: 표제란(Title Block) 인식, 도면 간 참조 관계 추출
- 기존 방식: 엔지니어가 도면을 보며 Excel에 수동 입력 (건당 30분~1시간) → Fabbit: 업로드 → AI 자동 추출 → 구조화 등록 (3분 이내 목표)

#### 2단계 — 구조화 BOM 관리 (데이터 축적)

AI가 추출한 데이터를 Excel로 내보내는 데 그치지 않고, **시스템 안에서 계층형 BOM 트리(Assembly-Sub Assembly-Part)로 관리**한다. Where-used(역추적), 리비전 추적, 도면-BOM 연계 검색이 가능하다.

#### 3단계 — AI가 먼저 발견하고 행동하는 제조 데이터 플랫폼 (중장기 비전)

> **협약 6개월 내 목표는 1~2단계(AI 추출 고도화 + 구조화 BOM 관리)에 집중**한다. 3단계는 인프라 준비가 완료된 상태이며, 협약 종료 시점에 프로토타입 착수를 목표로 한다.

축적된 부품·공급사·담당자·도면 데이터를 그래프 DB(Apache AGE)로 연결하여, AI가 데이터를 지속 분석하고 위험을 사전 감지하는 것이 중장기 방향이다 (Agentic AI). 팔란티어는 이 개념을 **Systems of Action**이라 부르며, 분석에서 끝나지 않고 실제 비즈니스 행동으로 연결하는 것이 핵심이다.

- **사전 감지**: 부품 변경 시 영향받는 제품·공급사를 자동 탐색하여 담당자에게 선제 알림
- **자율 행동**: 리스크 감지 → 대체 공급사 후보 추출 → 담당자에게 발주 초안 제시까지 자동화
- 인프라 준비 완료: `age_client`(그래프 DB), `ontology`(품목 분류), `llm_client`(자연어 쿼리) — 이 위에 프로토타입을 얹는 것이 협약 후 첫 과제

### 경쟁사 대비 우위

## 4. 팀 구성

> 평가: "기업 구성 및 보유 역량, 사회적 가치 도입"

### 대표자 전문성

컴퓨터공학 전공 후 **PLM 솔루션 기업**에서 BOM 관리 모듈을 개발하며 제조업 도메인 지식을 확보했다. 이후 SI 기업에서 프론트엔드/백엔드/인프라를 아우르는 풀스택 역량을 쌓았으며, 8명 규모 프로젝트의 기획/실행/일정 관리를 주도한 경험이 있다.

PLM 도메인 이해와 풀스택 개발 역량을 겸비하여, **MVP 설계부터 구현/운영까지 1인 실행이 가능**하다. 현재 도면·BOM AI 추출, PLM 코어 기능, 프론트엔드, 인프라를 단독 구축한 상태이다.

1. **중소 제조업 디지털 격차 해소**: PLM 도입률 4.4%인 중소 제조업에 AI 자동 추출 + 무료 Starter 플랜으로 도입 장벽을 제거하여, 대기업-중소기업 간 제조 데이터 관리 격차를 줄인다. → **Starter 플랜 영구 무료 유지**, 협약 기간 내 Starter 가입 기업 **10개사 이상** 목표.
2. **제조 현장 지식 보존**: 숙련공 퇴직 시 사라지는 암묵지(부품 이력, 공급사 노하우)를 시스템에 구조화하여 조직 지식으로 보존한다. → 파일럿 기업의 도면·BOM 데이터 **500건 이상** 시스템 등록 목표.
3. **경기도 제조업 경쟁력 강화**: 경기도 내 제조업체를 우선 타겟으로 파일럿을 운영하고, 바우처 연계를 통해 경기도 스마트공장 생태계에 기여한다. → 협약 기간 내 **경기도 소재 제조업 3개사 이상** 무상 파일럿 제공.
