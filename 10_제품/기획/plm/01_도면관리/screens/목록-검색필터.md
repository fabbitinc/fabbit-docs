# 검색 / 필터

목록 검색은 최신 승인본을 빠르게 찾는 데 목적이 있다. 문자열 검색과 상태
필터를 함께 쓸 수 있어야 한다.

- 검색 대상은 `도면번호`, `도면명`, `대표 연결 부품번호`다.
- 검색은 부분 일치 기준으로 동작한다.
- 상태 필터는 `Draft`, `In Review`, `Approved`, `Released`, `Obsolete`
  다중 선택을 지원한다.
- 변환 상태 필터는 `Pending`, `Processing`, `Completed`, `Failed`
  다중 선택을 지원한다.
- 유형 필터는 `2D / 3D`, `CAD / PDF / 이미지`를 조합한다.
- 기본 정렬은 `최종 수정일 내림차순`이다.
- 로딩 중에는 필터 패널은 유지하고 테이블만 스켈레톤으로 바꾼다.
- 빈 상태에서는 `검색 결과 없음`과 `새 도면 등록` CTA를 함께 노출한다.
- 오류 상태에서는 마지막 검색 조건을 유지하고 `다시 시도`를 노출한다.
