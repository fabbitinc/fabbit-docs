# 저장 상태

저장 상태는 사용자가 현재 무엇 때문에 저장하지 못하는지 바로 이해하게 해야
한다. 저장 중, 검증 실패, 변환 실패를 같은 영역에서 다룬다.

- 저장 중에는 `저장` 버튼을 비활성화하고 상단 진행 배너를 노출한다.
- 필수값 누락은 필드 하단 오류와 상단 요약 배너를 동시에 보여 준다.
- 도면번호 중복은 서버 응답을 받아도 필드 오류 위치를 유지한다.
- 저장 성공 후에는 상세 화면으로 이동하거나 같은 화면에 성공 토스트를
  노출한다.
- 상세 화면에서 변환이 실패하면 실패 코드는 유지되고 사용자는 `재처리` 또는
  `원본 다운로드`를 선택한다.
- 페이지 이탈 시 저장되지 않은 변경이 있으면 확인 모달을 띄운다.
