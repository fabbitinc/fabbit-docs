# 원본 파일

원본 파일 영역은 업로드 완료 파일을 도면 엔티티에 연결하는 역할을 한다.
실제 바이트 업로드는 파일 모듈이 끝낸 뒤 이 화면에서 선택한다고 가정한다.

- 신규 등록 시 원본 파일 선택은 필수다.
- 원본 파일은 `UPLOADED` 상태여야 한다.
- 지원하지 않는 확장자는 저장 전에 차단한다.
- 파일을 교체하면 `conversionStatus`는 즉시 `Pending`으로 돌아간다.
- 2D/3D 분류는 확장자를 기준으로 자동 판정하고 사용자는 읽기 전용으로 본다.
- `Released` 도면은 원본 파일 교체를 허용하지 않는다.
- 파일이 누락되면 `검토 요청`과 `배포`는 모두 막는다.
