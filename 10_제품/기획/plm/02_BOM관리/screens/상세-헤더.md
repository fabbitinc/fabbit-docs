# 헤더 / 루트 요약

상세 화면 상단은 "지금 어떤 부품을 어떤 방향으로 보고 있는지"를 명확히
보여 주는 영역이다.

- 루트 요약은 `GET /api/v1/parts/{partId}` 응답을 사용한다.
- 상단에는 `품번`, `품명`, `상태`, `현재 리비전`, `카테고리`를 노출한다.
- 진입 컨텍스트가 Where-Used면 초기 방향은 `REVERSE`다.
- 직접 자식/직접 부모 수는 `GET /api/v1/parts/{partId}/bom` 결과를 사용한다.
- 존재하지 않는 루트면 상세 화면 대신 `not found` 오류 상태로 전환한다.
