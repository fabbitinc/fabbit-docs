# 변경요청 편집 · 기본 정보

변경요청 편집은 `신규 등록`, `DRAFT 수정`, `SUBMITTED 수정` 세 상황을
처리한다. `MERGED`, `CLOSED`는 편집 대상이 아니다.

## 필드 규칙

| 필드 | 규칙 |
|------|------|
| 제목 | 필수, 500자 이하 |
| 본문 | TipTap JSON, 멘션 허용 |
| 원본 이슈 | 생성 시 기본 연결값으로 채움 |
| 연결 부품 | 복수 선택 가능 |

## 예외 케이스

- `MERGED/CLOSED` 변경요청 수정은 `INVALID_STATE`로 실패한다.
- 잘못된 TipTap 문서는 저장 실패다.
- 제목이 비어 있으면 저장과 제출 버튼을 모두 비활성화한다.
