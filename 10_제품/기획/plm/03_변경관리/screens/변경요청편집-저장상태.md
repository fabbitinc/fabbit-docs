# 변경요청 편집 · 저장 상태

- 기본 CTA는 `저장`, 보조 CTA는 `제출`, `취소`다.
- `저장`은 현재 상태를 유지하고 내용만 반영한다.
- `제출`은 저장 후 `DRAFT -> SUBMITTED` 전이를 연속 수행한다.
- `VALIDATION_ERROR`: 제목, 본문 구조, 연결 대상 오류를 필드 단위로 표시한다.
- `INVALID_STATE`: 화면 상단에 `현재 상태에서는 수정할 수 없습니다` 배너를
  노출한다.
- `FORBIDDEN`: 전체 폼을 읽기 전용으로 전환하고 상세 화면 복귀 CTA를 노출한다.
- `saving/submitting` 상태에서는 중복 클릭을 막는다.
