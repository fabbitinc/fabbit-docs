# 이슈 편집 · 기본 정보

이슈 편집 화면은 `신규 등록`과 `OPEN 상태 수정` 두 경우를 처리한다.
`CLOSED` 이슈는 편집 진입 대신 상세 화면 읽기 전용으로 유도한다.

## 필드 규칙

| 필드 | 규칙 |
|------|------|
| 제목 | 필수, 500자 이하 |
| 본문 | TipTap JSON, 멘션 허용 |
| 연결 부품 | MVP 직접 연결 대상 |

## 예외 케이스

- 닫힌 이슈 수정 요청은 `INVALID_STATE`로 실패한다.
- 잘못된 TipTap 문서 구조는 저장 실패다.
- 제목이 비어 있으면 저장 버튼을 비활성화한다.
