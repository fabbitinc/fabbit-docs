# 변경요청 상세 · 검토

- 검토 영역은 `개인 검토자`, `팀 검토자`, `리뷰 상태`, `리뷰 시각`을 노출한다.
- 개인 검토자에게만 `승인`, `반려` 버튼을 노출한다.
- 팀 검토자는 현재 배정 현황만 보여주고 리뷰 제출 버튼은 노출하지 않는다.
- `PENDING`은 회색, `APPROVED`는 완료 색상, `REJECTED`는 경고 색상으로 구분한다.
- MVP에서는 검토 결과와 별개로 `반영` 버튼이 남을 수 있다는 안내 문구를
  하단에 고정 노출한다.
- `empty`에서는 `검토자가 아직 지정되지 않았습니다` 문구를 노출한다.
