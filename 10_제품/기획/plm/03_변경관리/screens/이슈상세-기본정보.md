# 이슈 상세 · 기본 정보

이슈 상세는 접수된 변경 문제의 기준 화면이다. 제목, 본문, 상태, 연결
부품, 첨부파일, 라벨, 담당자를 한 번에 읽을 수 있어야 한다.

## 동작 규칙

- 제목, 상태, 작성자, 담당자, 연결 부품은 첫 화면에서 바로 보인다.
- 본문은 TipTap 렌더러로 읽기 전용 표시한다.
- 첨부파일은 파일명과 개수만 먼저 노출하고, 상세 다운로드는 클릭 시 연다.
- `CLOSED` 이슈는 화면 상단에 `종결됨` 배너를 표시한다.

## UI 상태

- `loading`: 기본 정보 블록 전체를 스켈레톤 처리한다.
- `error`: 상세 조회 실패 배너와 목록 복귀 CTA를 함께 노출한다.
- `read only`: 권한이 없어도 값은 모두 보이고 액션만 숨긴다.
