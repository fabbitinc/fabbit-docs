# 사이트 기획

이 문서는 Fabbit 대외 사이트 기획의 기준 문서다. 제품 기획 문서군 중에서도
획득과 전환을 담당하는 웹 페이지를 정의하며, 랜딩 페이지, 요금제 페이지,
데모/신청 흐름 같은 외부 진입 화면을 관리한다.

## 1. 사이트의 역할

- 제품을 처음 접한 방문자에게 Fabbit의 문제 해결 범위를 짧은 시간 안에 전달
- 초기 타겟 고객을 선별하고 파일럿/데모 리드를 수집
- 사업계획서와 영업 자료의 핵심 메시지를 웹 경험으로 일치시키기

## 2. 기획 원칙

- 첫 화면에서는 `PLM`, `MES`, `온톨로지`보다 고객 언어를 우선한다.
- 현재 구현된 기능과 계획 중인 기능을 명확히 분리한다.
- CTA는 `지금 결제`보다 `파일럿`, `데모`, `상담` 중심으로 설계한다.
- 제품 스크린샷, 정량 근거, 도입 대상 선별 정보를 함께 배치한다.
