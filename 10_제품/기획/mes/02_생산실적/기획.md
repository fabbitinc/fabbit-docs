# 생산실적 기획

이 문서는 작업지시 수행 결과를 기록하는 생산실적 기능을 정의한다. 목표는
양품, 불량, 작업시간을 간단히 입력해 일일 생산 현황을 가시화하는 것이다.

## 1. 문제와 목표

- 현장은 오늘 실제 생산량을 엑셀이나 수기로 기록하고 있다.
- 관리자는 종료된 작업의 실적을 당일 안에 확인하고 싶다.
- 이후 품질, 자재, 원가 분석의 기초 데이터가 필요하다.

## 2. 주요 사용자

- 작업자
- 현장 리더
- 생산관리자

## 3. 메뉴명과 진입 경로

실적 입력은 전문 용어보다 짧고 직접적인 메뉴명이 낫다.

| 구분 | 권장 명칭 | 비고 |
|------|-----------|------|
| 상위 메뉴 | `생산` | 공통 상위 메뉴 |
| 하위 메뉴 | `실적` | 메뉴는 짧게 |
| 화면 제목 | `생산실적` | 목록/상세에서 사용 |
| 기본 CTA | `실적 입력` | 행동 중심 문구 |

진입 경로 예시는 아래와 같다.

`생산 > 실적 > 생산실적 목록 > 실적 입력`

## 4. 핵심 시나리오

1. 작업자가 작업지시를 시작한다.
2. 작업 종료 후 양품 수량, 불량 수량, 작업시간을 입력한다.
3. 작업지시 상태가 자동 갱신된다.
4. 불량이 있으면 불량기록 기능과 연결한다.

## 5. 화면 기획

### 실적 입력 화면

- 참조 정보
  - 작업번호, 품목명, 계획 수량, 담당
- 입력 영역
  - 양품 수량
  - 불량 수량
  - 작업 시작/종료 또는 총 작업시간
  - 메모

### 작업지시 상세 내 실적 섹션

- 누적 양품 / 불량
- 마지막 입력 시간
- 입력자
- 연결된 불량기록 건수

### 관리자 조회 화면

- 기간별 실적 목록
- 품목별, 팀별, 담당자별 집계
- 계획 대비 실적 차이

## 6. 화면 예시

### 실적 입력 화면 예시

```text
[생산] / 실적

┌──────────────────────────────────────────────────────────────┐
│ 생산실적 입력                              저장              │
├──────────────────────────────────────────────────────────────┤
│ 작업번호: WO-240301   품목명: 본체 조립   계획 수량: 100     │
├──────────────────────────────────────────────────────────────┤
│ 양품 수량 [ 40 ]   불량 수량 [ 2 ]   작업시간 [ 03:20 ]      │
│ 메모      [ 조립 완료, 체결 불량 2건 ]                       │
├──────────────────────────────────────────────────────────────┤
│ 연결 액션: [불량 기록 작성] [임시 저장] [완료 처리]          │
└──────────────────────────────────────────────────────────────┘
```

### 관리자 조회 화면 예시

```text
생산실적 목록

┌──────────────────────────────────────────────────────────────┐
│ 기간 [오늘]  팀 [전체]  품목 [전체]  검색 [작업번호/품목명] │
├──────────────────────────────────────────────────────────────┤
│ 작업번호   품목명        양품   불량   작업시간   입력자      │
│ WO-240301  본체 조립      40      2    03:20      김현장      │
│ WO-240302  커버 가공      50      0    02:10      박리더      │
└──────────────────────────────────────────────────────────────┘
```

## 7. 플로우차트

```text
작업지시 선택
    ↓
실적 입력 화면 진입
    ↓
양품/불량/작업시간 입력
    ↓
검증
 ┌──┴───────────────┐
 ↓                  ↓
정상                계획 초과 / 누락
 ↓                  ↓
저장              경고 후 수정
 ↓
불량 수량 있음?
 ┌──┴───────┐
 ↓          ↓
예          아니오
 ↓          ↓
불량기록 연결   작업지시 상태 갱신
      \      /
       done 또는 계속 진행
```

## 8. 기능 기획

- 실적 등록
- 실적 수정 이력 관리
- 작업지시 상태 자동 연동
- 계획 수량 대비 진행률 계산
- 기간별 조회

## 9. 정책과 예외

- 양품 + 불량 > 계획 수량인 경우 경고
- 완료 처리 후 수정은 권한 제한
- 불량 수량 입력 시 불량기록 작성 유도

## 10. 데이터 연결

- 작업지시
- 불량기록
- 사용자 / 팀
- 향후 자재추적, 원가, 품질 데이터

## 11. MVP 범위

- 단건 실적 입력
- 양품/불량/작업시간 기록
- 작업지시 상태 연동
- 기본 집계

## 12. 후속 확장

- 부분 실적 여러 번 입력
- 교대별 실적
- 자동 이상 감지
- ERP 실적 송신
