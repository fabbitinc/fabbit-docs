# 경쟁사/대체재 비교

이 문서는 Fabbit의 경쟁 환경을 정리한다.
사업계획서 Solution 항목의 차별성 근거로 사용한다.

## 1. 비링커 클라우드 (최직접 경쟁)

Fabbit과 타겟·기능이 가장 겹치는 경쟁사. 2025년 8월 정식 출시.

| 항목 | 비링커 클라우드 |
|------|---------------|
| 가격 | Std 33,000원/월, Plus 44,000원/월, Premium 55,000원/월 |
| 과금 방식 | 플랜 기반 (사용자 수 기반) |
| AI 도면 인식 | **있음** — 비전 AI, 수기/레거시 도면 포함 |
| BOM 자동생성 | **있음** — 도면 → 품명·재질·수량·후처리 자동 추출 → Excel 변환 (3초) |
| AI 추출 횟수 | 500회(Std) / 1,500회(Plus) / 3,000회(Premium) |
| 스토리지 | 700GB ~ 3TB |

| 변경관리 (ECR/ECO) | **없음** |
| BOM 리비전 추적 | **없음** |

| 투자 | Seed+Pre-A 누적 10억+, 스트롱벤처스(미국VC), 세아그룹 VNTG, 한투AC |
| 기타 | 글로벌 팁스 선정 (3년 최대 6억), ISO 27001, 북미 진출 준비 |

| 고객 사례 | 코아스(KOAS) 도입 후기 공개 (2025.07) |

| 글로벌 | 미국 법인 설립 추진, 북미 고객 확장 |

> [!warning] Fabbit과의 핵심 차이
> 비링커는 **도면 분석 도구**(AI extraction → Excel)이며, **PLM이 아님**.
> 도면에서 BOM 데이터를 뽑아주지만, 뽑은 후의 관리(구조화, 변경추적, 리비전)는 사용자 몫.
> Fabbit은 추출 + 관리(lifecycle)를 하나의 시스템에서 해결하는 것이 차별점.

> [!warning] 비링커의 PLM 확장 가능성
> 비링커가 공식적으로 "PLM 확장" 로드맵을 발표한 적은 없음.
> 그러나 "소프트웨어와 제조 서비스를 결합한 글로벌 제조 플랫폼" 목표를 언급하고 있으며,
> "도면·제조 데이터 관리 SaaS 운영" + 매뉴팩처링 결합 방향으로 사업을 확장 중.
> **Fabbit 관점**: 비링커가 PLM 영역이 아닌 매뉴팩처링(위탁생산)으로 확장 중이므로, PLM 영역에는 여전히 진입 기회가 존재함.

## 5. 현행 방식 (엑셀/파일서버/종이)

중소 제조업의 **실질적 경쟁 상대**. PLM 도입률 4.4%이므로 95%+가 이 방식.

### 엑셀 BOM 관리의 문제점

- **버전 충돌**: 부서별로 각자 복사본 사용 → SSOT 부재
- **수동 입력 오류**: 연간 780시간 엔지니어링 시간 낭비, $39K 인건비 (중견기업 기준)
- **수량/사양 오류**: 제품 원가의 3~5%가 잘못된 수량 발주로 낭비
- **변경 반영 지연**: 설계변경(ECO) 프로세스가 수일~수주 소요
- **확장 불가**: 품목 수 증가 시 관리 한계

### 파일서버/공유폴더 도면관리의 문제점

- **검색 시간**: 엔지니어가 업무시간의 최대 30%를 부품/도면 정보 검색에 소비
- **버전 혼선**: 최신 도면 여부 확인 불가 → 구버전으로 생산/발주
- **접근 제어 없음**: 수정 이력 추적 불가
- **협업 불가**: 외부(협력사) 공유 시 이메일/USB 기반

## 6. Fabbit 포지셔닝

### 경쟁 구도 요약

```
                  도면 AI 인식
                      ↓
  ┌─────────────────────────────────────────┐
  │           비링커 클라우드                   │ ← AI 추출 도구
  │   도면 → AI → Excel BOM (여기서 끝)        │
  └─────────────────────────────────────────┘
                      ↓ 추출 후 관리는?
  ┌─────────────────────────────────────────┐
  │              Fabbit                      │ ← AI + PLM
  │   도면 → AI → 구조화 BOM → 관리·추적·변경   │
  └─────────────────────────────────────────┘
                      ↓ 본격 PLM이 필요하면?
  ┌─────────────────────────────────────────┐
  │     코너스톤 / 싱글톤 / JK-PLM 등          │ ← 기존 PLM (AI 없음)
  └─────────────────────────────────────────┘
```

> [!info] Fabbit 핵심 차별점 (수정)
> ~~도면 AI 인식은 비링커가 이미 상용화 — 더 이상 유일한 기능이 아님.~~
> 1. **AI 추출 + PLM 관리의 결합**: 비링커는 추출(도면→Excel)까지, Fabbit은 추출 후 구조화·추적·변경관리까지 하나의 시스템에서 해결
> 2. **회사 단위 과금** (사용자 수 무관): 비링커(플랜별)·국내 PLM(인당) 대비 예측 가능한 비용
> 3. **즉시 도입 + 낮은 학습곡선**: 엑셀에서 바로 전환 가능
> 4. **기존 PLM 대비 AI 진입점**: 코너스톤·싱글톤 등 기존 PLM에는 없는 AI 자동화

> [!warning] 시장 검증 시사점
> 비링커의 투자 유치(10억+)와 글로벌 팁스 선정은 **AI 도면 인식 시장의 존재를 입증**한다.
> 단, 비링커는 PLM 영역으로 확장하지 않고 위탁생산(매뉴팩처링)으로 사업을 넓히고 있어,
> PLM 영역은 여전히 진입 기회가 있음.
