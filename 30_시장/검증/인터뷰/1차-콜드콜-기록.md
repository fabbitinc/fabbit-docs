# 1차 콜드콜 기록 (2026-02-25 ~ 02-27)

화성 소공인 업체 대상 콜드콜 기반 인터뷰.
목적: PLM/BOM 관리 현황 파악 및 시제품 사용 의향 확인.

## 1. 전체 현황

| 항목 | 수치 |
|------|------|
| 총 콜 수 | 36건+ |
| 통화 성공 | ~15건 |
| 인터뷰 성공 | 6건 (17%) |
| 시제품 의향 | 1건 (3%) |

### 대상 프로파일

| 항목 | 값 |
|------|---|
| 지역 | 경기도 화성시, 김포시, 시흥시 |
| 업종 | 금형, 기계제작, 레이저가공, CNC, 반도체장비 등 |
| 규모 | 대부분 영세기업~소기업 (1~9명 추정) |
| 방식 | 콜드콜 (전화) |
| 컨택 대상 | 인터넷 수집 → 대부분 CS 담당자 연결 |

## 2. 인터뷰 상세 (성공 6건)

### #11 동광기계

| 항목 | 값 |
|------|---|
| 업체명 | 동광기계 |
| 도메인 | 기계제작 (압출기/분쇄기/파쇄기) |
| 위치 | 경기도 화성시 봉담읍 |
| 날짜 | 2026-02-25 |

**Q&A:**
1. PLM / 도면관리 + BOM 관리 소프트웨어를 사용하시나?
   - 영세기업이라 사용하지 않음
2. 그렇다면 자재 관리는 어떻게 하시나?
   - 기업별로 폴더를 만들어서 관리함
   - 영세기업이라 관리할 capa가 나오지도 않음
3. 관리하기 용이한 솔루션을 만들고 있는데 무료 시제품을 사용하실 생각이 있으신가?
   - 곧 폐업한다고 함
